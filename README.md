# Ansible Collection - xhiroga.roles
[@xhiroga](https://github.com/xhiroga)'s ansible roles.


## Prerequisites

Install [Poetory](https://python-poetry.org/).

```shell
brew install ansible
poetry install
```

## Scaffolding

```shell
make -C ./roles init ROLE_NAME=${ROLE_NAME}
```
